如何使用Hugo创建网站

2018-01-02

Hugo是一个方便快捷的网站静态页面生成工具，通过该工具，可以将MARKDOWN书写的页面转化为网站页面。

Hugo入门

安装

如下已macOS上安装为例，其他操作系统安装方式可参考官方文档。

$ brew install hugo

创建站点

$ hugo new site /path/to/site

生成站点目录结构如下：

  • archetypes/
  • content/
  • layouts/
  • static/
  • themes/
  • config.toml

创建页面

$ cd /path/to/site/content
$ hugo new about.md

生产文件内容如下：

---
date: 2018-01-02T22:40:41+08:00
draft = true
title = "about"

---

使用Markdown来写正文内容

安装模版

Hugo有丰富的主题，可以从Hugo官方主题进行下载。

$ cd /path/to/site/themes
$ git clone https://github.com/spf13/hyde.git

调试

通过下面命令，Hugo会默认启动一个本地站点：http://localhost:1313.

hugo server --theme=hyde --buildDrafts

Hugo部署

如下是已Git Pages作为部署环境：

创建GitHub项目

在GitHub上创建一个仓库（Repository），名字为“yourname.github.io”(其中yourname为Hithub的用户名).

生成静态页面

通过如下命令可以在/path/to/site/public下生成静态页面。

hugo --theme=hyde --buildUrl=“yourname.github.com”
  • 注：如果不能生成静态页面，请检查并注释MD文件中“draft = true”。

提交静态页面

将/path/to/site/public下所有静态页面拷贝到新建的仓库yourname.github.io下。 很快就可以从https://yourname.github.io上访问到新上传的页面了。

$ cd public
$ git init
$ git remote add origin https://github.com/yourname/yourname.github.io.git
$ git add -A
$ git commit -m "first commit"
$ git push -u origin master

修改自定义域名

  • 在GitHub的仓库yourname.github.io下创建CNAME文件，并添加如下域名：
yourdomain.com
www.yourdomain.com
  • 进入域名服务商控制台，对域名增加A记录（A Record）,其中记录址需要替换为yourname.github.io的真实IP。
主机记录 类型 记录值
www A 127.0.0.1
@ A 127.0.0.1
